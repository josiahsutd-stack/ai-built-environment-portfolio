# Resume Project Bullets: General AI

- Built a multimodal visual QA assistant with image upload, VLM provider abstraction, structured JSON extraction, confidence scoring, uncertainty messaging, and a local mock inference path.
- Developed a planner-executor research agent with local RAG, tool calling, citation tracking, memory persistence, human approval checkpoints, and inspectable agent traces.
- Created a VLA-inspired embodied-agent simulator that maps language instructions and grid-world state into safe action sequences with episode replay and invalid-action safety checks.
- Implemented reinforcement-learning simulations for warehouse inventory control and dynamic pricing with custom environments, baseline policies, reward shaping, and policy evaluation.
- Built a computer-vision defect detection lab with synthetic image generation, validation metrics, example predictions, a model card, Streamlit demo, and FastAPI endpoint.
- Created an LLM evaluation and guardrails platform for prompt-injection detection, citation coverage, structured-output validation, and regression-style eval reporting.
- Built an MLOps model-serving demo with synthetic data generation, scikit-learn model training, FastAPI prediction schema, drift detection, model metadata, Docker, and tests.
- Developed a recommender-system ranking engine with synthetic interactions, popularity baseline, content-based retrieval, precision@k, NDCG@k, API endpoint, and explanation scores.
- Built a time-series forecasting and anomaly-detection project with synthetic API traffic, moving-average backtesting, Isolation Forest anomaly detection, alert counts, and evaluation metrics.
- Created a compute-aware fine-tuning and LoRA lab with dataset generation, validation, train/validation splits, LoRA config, mock trainer, and model-card limitations.
